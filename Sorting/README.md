### Algorithms pseudocodes:  

#### Insertion sort:  
> InsertionSort(array)  
>   for i = 2 to length(array)  
>      value <-- array(i)  
>      j <-- i - 1  
>      while j > 0 and array(j) > value  
>         array(j + 1) <-- array(j)  
>		   j--  
>      arr[j + 1] <-- value
